Welcome to the Omega-X Common Semantic Data Model Website

Description

Welcome to the Common Semantic Data Model Documentation. The CSDM is a modular ontology designed by EDF Team as part of the Work Package 4 of the Omega-X European project, to support semantic interoperability within the Omega-X energy dataspace. The CSDM has been developed following two ontology engineering methodologies: Agile Interaction Based Methodology for Energy Data Spaces (AIME) and ACIMOV for tooling. Here, you will find detailed information about each module, including documentation and source links. Use the table below to navigate to the specific module documentation and source files.
